# InsightOS

InsightOS is an AI-powered document intelligence agent built using Claude Projects.

## Overview

InsightOS analyzes multiple uploaded documents (PDF, Word, Excel) together instead of individually. It extracts key information, identifies tasks, deadlines, risks, and generates executive dashboards with actionable recommendations.

## Features

- Multi-document understanding
- Cross-document reasoning
- Executive summaries
- Task extraction
- Deadline detection
- Risk analysis
- Executive dashboard generation
- Strategic recommendations

## Technologies

- Claude Projects
- Prompt Engineering
- AI Agents
